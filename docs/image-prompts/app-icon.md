# App Icon — Image Generation Prompt

## Target file
`assets/icon.png`

## Required size
**1024 × 1024 px** source → export final at **512 × 512 px**

## Format
PNG, transparent background

## Prompt
> Minimal Scandinavian smart home app icon for a lawn soil temperature optimization app, vibrant green grass blades forming a circular temperature gauge, subtle soil layers underneath, soft weather-inspired gradients, modern IoT aesthetic, premium smart gardening branding, clean vector style, rounded friendly shapes, highly recognizable at small sizes, dark and light mode compatible, centered composition, transparent background, no text, no watermark, app-store quality

## Style notes
- Primary colour: rich grass green `#4CAF50` → `#2E7D32`
- Accent: warm amber sun `#FDD835`, cool earth brown `#8D6E63`
- Background: **transparent** (do not use white fill)
- Rounded-square safe zone: keep important artwork within the inner 80 % of the canvas
- Should look sharp at 32 × 32 px and stunning at 512 × 512 px

## Export steps
1. Generate at 1024 × 1024
2. Export as `assets/icon.png` at 512 × 512, PNG-8 or PNG-24 with alpha
3. Optionally keep the 1024 source as `assets/icon-source.png`
