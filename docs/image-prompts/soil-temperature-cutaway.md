# Soil Temperature Cutaway — Image Generation Prompt

## Target file
`assets/images/soil-temperature-cutaway.png`

## Recommended size
**1200 × 800 px**

## Format
PNG, transparent or white background

## Prompt
> Cutaway illustration of healthy lawn soil layers, visible underground temperature gradients, moisture indicators, root zone visualization, realistic grass texture, clean educational style, transparent background, premium environmental visualization

## Style notes
- Left-to-right or top-to-bottom cross-section view
- Distinct layers labelled by depth (approximate):
  - 0 cm  — grass surface / thatch
  - 6 cm  — upper root zone (sensor depth)
  - 10 cm — mid root zone
  - 30 cm — deep root zone
- Temperature gradient colour coding (cool blue → warm amber)
- Moisture droplets or water table hint in deeper layers
- Root tendrils visible reaching into soil

## Usage
- Documentation / README illustration
- Could be used as a background in marketing materials
- Optionally embedded in the Homey app settings page

## Export steps
1. Generate at 1200 × 800
2. Save as `assets/images/soil-temperature-cutaway.png`
